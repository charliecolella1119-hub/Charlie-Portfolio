Project Case Study

Molecular Dynamics Simulator

Scientific Model

The simulator models Lennard-Jones particles with a short-range repulsive core and longer-range attraction. Periodic boundaries and the minimum-image convention make the simulation box behave like a small repeating material sample.

Numerical Method

Motion is integrated with velocity Verlet, a common molecular dynamics method that keeps position and velocity updates stable enough for interactive energy, pressure, and temperature inspection.

Analysis Tools

The browser demo lets visitors test the core simulation immediately. The full C++ app compares particle speeds to a Maxwell-Boltzmann distribution, computes a normalized radial distribution function, tracks pressure-temperature samples, and exports thermodynamic data to CSV.
